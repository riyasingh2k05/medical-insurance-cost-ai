# Medical Insurance Cost Prediction using XGBoost

## 📌 Project Overview
This project predicts individual medical insurance costs based on lifestyle and demographic factors. By leveraging advanced machine learning, the model provides highly accurate estimates for healthcare premiums.

## 🚀 Key Achievements
* **Accuracy:** Achieved an **R² score of 88.47%**.
* **Model:** Implemented **XGBoost Regressor**, an ensemble learning technique.
* **Insights:** Identified **Smoking Status**, **BMI**, and **Age** as the top three drivers of insurance costs.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-Learn, XGBoost, Seaborn, Matplotlib
* **Environment:** Google Colab / Jupyter Notebook

## 📊 Key Findings
- Smokers are charged significantly higher premiums regardless of age.
- High BMI combined with smoking creates a exponential increase in costs.
